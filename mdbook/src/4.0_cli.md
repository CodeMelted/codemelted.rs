# 4.0 Command Line Interface (CLI)

<mark>Beginning works of the codemelted CLI Native command.</mark>

*Commands to validate with Invoke-CodeMelted.psm1 binding to get the full terminal stack in pwsh.*
- Installed via `cargo install codemelted`
- To upgrade just the native CLI `cargo upgrade -p codemelted`
- To upgrade all items, `cargo upgrade`

```
================================================================
codemelted (CLI Native Command)
================================================================

ABOUT: Provides a rust compiled native executable that supports
  Linux, Mac, and Windows terminals. Allows for utilizing the
  native scripting (BAT, shell) for the given operating system.
  The power with this CLI is the actions implemented are the
  same regardless of the operating system. Consider installing
  the codemelted-cli PowerShell module to get a full fledged CLI
  experience. That module extends the codemelted CLI
  functionality to get a true scripting terminal experience.

SYNTAX: codemelted [action] [params]

USAGE:
  --console-alert [message]
      Pauses execution while reporting a message.
  --console-confirm [message]
      Provides a confirmation prompt via STDOUT with the answer
      being written to STDOUT as true / false.
  --console-choose [message] [choices]
      Provides a selection menu of the CSV choices waiting for
      a valid selection from the user with the chosen index
      written to STDOUT.
  --console-password [message]
      Provides a password prompt writing out the password to
      STDOUT.
  --console-prompt [message]
      Provides an input prompt writing out the answer to
      STDOUT.
  --console-write [message]
      Writes [message] to STDOUT with no newline.
  --console-writeln [message]
      Writes [message] to STDOUT with a newline.
  --help Prints this help system.

WEBSITE: https://codemelted.com/developer/
```
