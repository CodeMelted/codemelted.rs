# 5.0 Raspberry Pi

<center><img style="max-width: 480px;" src="models/logo-the-planned-trip.png" /></center>

Welcome to the *CodeMelted Raspberry Pi* project. This project will make use of the `codemelted.rs` project to design and build an IoT system that runs on a Raspberry Pi Single Board Computer. It will provide a REST API endpoint to facilitate the capture of picture / videos for sky watching and what is deemed Jeep Adventure.

The Jeep Adventure idea was born out of the idea to travel within a Jeep Wrangler to Prudhoe Bay, Alaska. This encompasses roughly a three week round trip journey. The [6.0 Original Prototype](#61-original-prototype) setup Raspberry Pi infotainment system with picture capture capabilities. Further fleshing out of this idea include now adding a sky watching capture capability to compliment the complete system.

## 6.2 The New Design

The new design will focus on two hardware form factors. The first being a Jeep Wrangler dash cam. The second is as a dedicated sky watcher. The following *Acceptance Criteria* identify what the goals of the new design.

**Acceptance Criteria:**

1. Provides the ability to capture still images every X seconds for Y time storing those images on the local SD card / usb drive.
2. Provide the ability to capture video for X time storing that video on the local SD card / usb drive.
3. Provide the ability to remote control the device to perform the following:
   1. Take still images as specified in AC 1.
   2. Take videos as specified in AC 2.
   3. View a current live video stream.
   4. Get current device health metrics (CPU, hard drive space, GPS, etc.).
   5. Provide ability to manage the device to perform updates and download currently captured images / video.
4. Provide a diagnostic Terminal User Interface (TUI) to provide a local ability to accomplish the tasks identified in AC #3.
